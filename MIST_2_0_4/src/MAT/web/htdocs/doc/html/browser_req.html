Browser Requirements

The MAT UI is developed using the most recent version of Firefox. You should use it. We try to code using Web standards, but some of the JavaScript objects we need were not implemented in all browsers as of this writing. Here's what we know about other browsers:

  • Firefox 2: known to have a significant CSS bug involving scrollbars which renders it essentially unusable. We doubt that anyone will encounter this browser in the wild anymore.
  • Safari, versions 3 and later: frequently works. We do not test regularly under Safari, but we have internal users who use it reliably.
  • Chrome: We do not test at all under Chrome, but we have internal users who use it reliably.
  • Opera: unknown.
  • IE6-8: known not to work (DOM Range object does not exist, so span annotation can't be done).
  • IE9: unknown.

It is extremely unlikely that any version of IE will ever be supported.
