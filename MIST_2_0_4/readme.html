The short version

Welcome to the MITRE Identification Scrubber Toolkit (MIST), built on top of the MITRE Annotation Toolkit (MAT). This documentation is best viewed with Firefox. Please proceed.

The long version

The system you're about to learn about consists of a core application along with some plugins we call tasks, and it provides a facility for you to add tasks if you wish. The documentation you're about to examine has been statically generated, and incorporates the documentation associated with the tasks that were known when the documentation was generated.

Our system includes a Web server which makes available a dynamic version of this documentation, which incorporates the documentation associated with the tasks it knows about at runtime. If you've added or removed tasks since you installed this package (and if you do that, you'll know), the dynamic documentation might differ from the static documentation, and will be more up-to-date.

Now that you've been warned that this version of the documentation might get stale, please proceed.
